# Consumption API GOV - Emendas Especiais

This project is a Python-based API for accessing and processing data related to "Emendas Especiais" from government sources.

## Features

- Fetches and processes data from government APIs
- Designed for transparency and data analysis
- Developed and maintained by the SCGE organization
